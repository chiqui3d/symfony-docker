# Docker compose for Symfony
A Full Stack configuration for Symfony with Docker 
# What's included?

- Nginx with Brotli
- PHP-FPM 7.2
- Mysql 8
- PHPMyAdmin

# Extra's

- virtual hosts
- elk (optional, currently commented in the compose file)
- composer
- brotli
- gzip
- cron
- opcache
- intl
- apcu
- wkhtmltopdf
- jpegoptim
- optipng
- pngquant
